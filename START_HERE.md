## 🎓 Code Overview

### Authentication Flow
```
Page loads → Check localStorage → 
  If user: Show app → If not user: Show login/signup →
    User fills form → Validate → Create account →
      Save to localStorage → Set session →
        Show dashboard
```

### Data Isolation
```
When user creates content:
→ Add userId to every item
→ When querying: Filter by current userId
→ Result: Each user sees only their data
```

### Premium Check
```
On login: Check tier and expiry date →
  If expired: Downgrade to free →
    If free: Limit to 3 clients →
      If premium: Show unlimited
```

## 🐛 Troubleshooting

**App won't load?**
- Check file path
- Enable JavaScript
- Check browser console (F12)

**Login not working?**
- Password must be 6+ chars
- Check email/phone format
- Try clearing browser cache

**Data not saving?**
- Check localStorage isn't full
- Reload page
- Check DevTools → Application → Storage

**Can't add 4th client?**
- That's correct! Free tier is 3 clients
- It's a feature, not a bug! 😄
